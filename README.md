# DEPI-Final-Project
# 📊 Store Sales Data Analysis and Forecasting Dashboard

## 📌 Project Idea
This project focuses on analyzing a store sales dataset to extract meaningful insights that support business decision-making.  
The project includes data cleaning, exploratory data analysis, and identifying patterns in sales performance across products, categories, regions, and time.  

In addition, forecasting techniques will be applied to predict future sales trends. The final outcome will be an interactive dashboard and a detailed report that helps managers understand sales behavior and plan future strategies.
